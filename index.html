usar sem internet (modo offline)

Exporte os inscritos como JSON da planilha e cole abaixo para usar o app sem internet no dia do evento.

como configurar
Opção 1 — Web App (recomendado):
1. No Apps Script, adicione a função doGet() do arquivo v6_doGet.js
2. Publicar → Implantar como Web App → acesso "Qualquer pessoa"
3. Copie a URL e cole acima

Opção 2 — CSV público:
1. Na planilha, vá em Arquivo → Compartilhar → Publicar na Web
2. Selecione aba "Inscritos" → formato CSV
3. Copie a URL e cole acima

Opção 3 — JSON offline:
1. No Apps Script, execute exportarJSON()
2. Copie o JSON gerado e cole no campo acima
